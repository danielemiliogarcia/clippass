Clippass Settings

Source code available here: https://github.com/danielemiliogarcia/clippass

Clippass is a Chrome extension designed to securely manage and decrypt passwords that have been encrypted using a PGP key. The extension integrates with your operating system to perform the decryption process because the PGP key and related operations are managed by your system and not the browser.

The main idea is to use this extension with this password manager https://www.passwordstore.org/ using the -e feature that I added. See This merge request

To enable this functionality, the extension requires a native system host that can securely interact with the clipboard and decrypt the encrypted password. This native host facilitates communication between the browser and the system-level script responsible for decryption.

To set up the native system host, you must download and run the installer provided below. This installer will:

  • Add a script to your system to decrypt passwords from the clipboard.
  • Register the native messaging host to allow communication between the extension and your system.

Once the native system host is installed, the extension will work seamlessly to decrypt passwords and input them securely into the focused field on your browser.

USAGE: To use it: copy your encrypted password to the clipboard (see password-store [pass] -e option), then focus the text or password field , then click the extension icon and the password will be inserted in plaintext. Do not forget to touch your yubi key if you are using one.

Click the "Download" button then, its a bash script, then inspect it if you want, and run it in your terminal.

Note: The installation process requires running a script on your system. This is necessary for the extension's functionality and is performed securely.
